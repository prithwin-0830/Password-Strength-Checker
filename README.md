# Password Strength Checker

## Overview
This Python application evaluates the strength of user passwords based on multiple security criteria.

## Features
- Checks password length
- Verifies uppercase and lowercase letters
- Detects numbers and special characters
- Provides improvement suggestions
- Classifies passwords as Weak, Moderate, Strong, or Very Strong

## Technologies Used
- Python
- Regular Expressions (re)

## Usage

```bash
python password_strength_checker.py
```
